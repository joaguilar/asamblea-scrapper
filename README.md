# asamblea-scrapper

# Installation instructions

## Prerequisites

1. Brew Packages

```
brew install pkg-config
brew install hdf5
export HDF5_DIR=/opt/homebrew/opt/hdf5@1.13                                                    
```

2. Python Packages

```
python3 -m pip install tensorflow-macos
python -m pip install tensorflow-metal
```
